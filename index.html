Спринт: один день тренировок

Здесь я расскажу про один день из своих тренировок

Разминка

Одна из важных частей любой тренировки - подготовка в нагрузке. Привести организм в тонус, разогреть связки и мышцы.

  • Раскатывание на валике
  • Растягивание основных мышц ног
  • Барьеры: 2х передом / 2х сбоку согнутой ногой / 2х сбоку прямой ногой / два вперед + один назад / мельница
  • Беговые упражнения: высокое бедро / чередование / прямые ноги / боковые / перекаты / выпрыгивания

Основная работа

  • Бег: дисктонт 100-80-60-40
  • Зал: 3х присед 50кг + выпрыгивания на маты / 3х выпрыгивания на две ноги с гирей + махи одной ногой / 3х румынская тяга на одной ноге с опорой
